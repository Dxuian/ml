"PVDATA_sample" contains sample of spatio-temporal PV monitoring data of the University of Queensland, from 20180721 to 20180817. 
"Weather_sample" contains sample weather data of the greater Brisbane area from 20180721 to 20180817. 
"Code_sample" contains DEMO code for the BILST model for showing the concept.

We are planning to establish an open platform for the PV spatio-temporal forecasting. The above folder will be updated in the future and new data will be added as well.
